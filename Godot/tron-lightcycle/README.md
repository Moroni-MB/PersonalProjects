# Tron Game Lightcycle

A 2D grid-based Tron-like game made with Godot where the player controls a light cycle, leaving a trail behind. The game includes movement, collision detection, and AI-controlled enemies. Players can customize their character color, and enemies follow simple AI to avoid walls and trails while trying to trap the player.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open Godot and load the project folder.
2. Make sure the main scene (`MainMenu.tscn`) is set as the active scene.
3. Press "Play" or run the project to start the game.

Instructions for using the software:

1. Use arrow keys to move your character.
2. Avoid crashing into the edges of the screen, trails, or enemies.
3. Customize your character color in the menu settings if desired.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* [Godot Engine 4.x](https://godotengine.org/)
* A code editor like [VS Code](https://code.visualstudio.com/) or Godot’s built-in script editor
