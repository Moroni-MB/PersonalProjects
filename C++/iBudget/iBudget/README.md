# iBudget

This project is a simple command-line budget manager written in C++. The program allows users to track income and expenses, view all recorded transactions, calculate their current balance, and save or load transactions from a file. The program demonstrates the use of conditionals, loops, functions, classes, and a Standard Template Library (STL) data structure. It also includes file input and output to preserve transaction data between program runs.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open the project in Xcode.
2. Build the program by clicking the Run button or pressing Command + R.
3. The program will compile and run in the console window.

Instructions for using the software:

1. Choose an option from the menu displayed in the console.
2. Enter transaction information such as the amount and category when prompted.
3. View transactions or the current balance, and select **Save & Exit** to store the data before closing the program.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Xcode (latest version available on macOS)
* C++ Compiler (Clang, included with Xcode Command Line Tools)
* macOS operating system
