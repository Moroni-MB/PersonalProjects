# Workout Generator (TypeScript)

A web-based workout generator built with TypeScript that creates personalized weekly workout plans based on a user's fitness goal, training frequency, and available equipment. The application demonstrates strong TypeScript fundamentals including typed variables, functions, arrays, classes, tuples, inheritance, and exception handling. The workout plan is dynamically rendered in the browser using a clean card-based UI.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Node.js (v18+ recommended).
2. Run `npm install` to install project dependencies.
3. Run `npx tsc` to compile TypeScript to JavaScript.
4. Run `npm run dev` to start the lite-server.
5. Open `http://localhost:3000` in your browser.

Instructions for using the software:

1. Select your fitness goal (Strength, Hypertrophy, or Fat Loss).
2. Enter the number of workout days per week.
3. Select available equipment.
4. Click the Generate button.
5. View your personalized workout plan displayed as organized workout cards.
6. If invalid input is entered, an error message will display.
